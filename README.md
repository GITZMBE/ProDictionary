## Installation

To get started with ProDictionary, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/ProDictionary.git
   cd ProDictionary
   npm install
   npm run dev

2. **Install dependencies**:
  ```bash
  npm install
  ```

3. **Run project**:
  ```bash
  npm run dev
  ```
